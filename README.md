## ğŸ› ï¸ Installation

Red URL Saver is currently available for manual installation. Follow these simple steps to add it to your browser:

1.  **Download the Source Code**:
    *   Clone this repository:
        ```bash
        git clone https://github.com/Rishabh157/Red-URL-Saver.git
        ```
    *   Or [Download ZIP](https://github.com/Rishabh157/Red-URL-Saver/archive/refs/heads/main.zip) and extract it.

2.  **Open Chrome Extensions**:
    *   Type `chrome://extensions/` in your address bar and press Enter.

3.  **Enable Developer Mode**:
    *   Toggle the switch in the top-right corner labeled **Developer mode**.

4.  **Load the Extension**:
    *   Click **Load unpacked**.
    *   Select the folder where you downloaded **Red URL Saver**.

5.  **Pin It**:
    *   Click the puzzle piece icon in your Chrome toolbar and pin **Red URL Saver** for quick access.
